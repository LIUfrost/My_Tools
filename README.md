# 一.网站资源
## 1.综合资源库
- [fmhy](https://fmhy.net):此网站整合多种资源，包括多种AI大模型，开发者网站，语言学习网站等等，可以说应有尽有了。
- [Z-Library](https://zh.z-library.sk):全球最大免费书籍网站之一，支持免费下载。
- [Github app](https://github.com/rainxchzed/Github-Store):跨平台github应用商店（推荐页比自己在github上瞎找好很多）
- [Droid-ify](https://droidify.app):开源应用平台
## 2.CS资源库
- [CS自学指南](https://csdiy.wiki)：来自北大信科大佬的计算机自学指南，本人自学大多跟着它的指导，最推荐的自学网站。
- [Github](https://github.com):世界上最大的同性交友平台（雾），既然已经看到这了，含金量不必多说。
- [Stack Overflow](https://stackoverflow.com):程序员的知乎，大佬云集之地（但IP纯度似乎要求过高？登了好久没登上去...）
- [hellogithub](https://hellogithub.com):定期分享github优质项目，并有精心分类和详细介绍（另有同名公众号）
- [hugging face](https://huggingface.co):著名人工智能模型与数据集的托管与分享平台，支持下载和在线运行AI模型（deepseek本尊便在此开源）
- [Advent to Code](https://adventofcode.com):趣味编程解密网站，每天全球几十万程序员同时等新题。很多大厂工程师、算法高手、大学教授也在玩，是程序员节日一样的活动。
- [Coursera](https://coursera.org):全球最大在线学习网站（世界版MOOC），众多名校，大厂在此开设课程（另有同名软件，已放仓库），跟着CS自学指南自会找到课程。
- [Leetcode](https://leetcode.com):力扣，“老少皆宜”的编程刷题网站，无论是数据结构/算法初学者，还是准备面试的程序员，都可以获得很大帮助（同名软件自行应用商店下载）
- [AI咨询日报](https://ai.hubtoday.app/):界面较简单，但信息有效。
- [BestBlog](https://bestblogs.dev):Github上2.3k项目，分享计算机最新消息，内容全面，信息源多样，功能丰富。
- [Hextra](https://ai.hubtoday.app):简单的AI日报。
- [zread](https://zread.ai):将Github仓库的URL导入即可分析该仓库，功能强大。
## 3.娱乐资源库
#### 番&漫
- [animeko](https://github.com/open-ani/animeko):集找番、追番、看番的一站式弹幕追番平台。界面简介美观，内容全面（反正我追的都有）
- [mihon](https://github.com/mihonapp/mihon):安卓漫画阅读神器，支持大量图源
#### 新闻
- [NewsNow](https://newsnow.busiyi.world):优雅地阅读实时且最热门的新闻,实时更新热门新闻，一键浏览方便快捷
- [TrendRadar](https://github.com/sansan0/TrendRadar): AI舆情监控助手与热点筛选工具。聚合多平台热点 + RSS 订阅，支持关键词精准筛选。（部署较复杂）
#### 音乐
- [musicdl](https://github.com/CharlesPikachu/musicdl):轻量级无损音乐下载器，支持网易云等大量主流音乐平
#### 其他
- [SillyTavern酒馆](https://github.com/SillyTavern/SillyTavern):面向高级用户的LLM前端，跟AI为所欲为(doge)
- [chatlab](https://github.com/hellodigua/ChatLab):聊天记录分析工具，想知道某人在微信上对你发过最多的消息是什么吗？
## 4.效率资源库
- [简悦](https://github.com/Kenshin/simpread):更好的网页浏览和阅读，支持将网页内容一键导入Obsidian中
- [open-interpreter](https://github.com/openinterpreter/open-interpreter):在终端运行大语言模型，代理电脑！比如让AI帮你整理文件，控制浏览器...
- [Catime](https://github.com/vladelaina/Catime):一个小巧（803KB）但强大的纯 C 计时器。支持时钟、倒计时、秒表、番茄钟，以及完全可自定义的托盘动画（GIF、CPU/内存%）💘
- [Localsend](https://github.com/localsend/localsend):一个开源的跨平台AirDrop替代方案,传输文件超级好用！
